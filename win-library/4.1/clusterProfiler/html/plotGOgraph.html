plotGOgraph {clusterProfiler}R Documentation

plotGOgraph

Description

plot GO graph

Usage

plotGOgraph(
  x,
  firstSigNodes = 10,
  useInfo = "all",
  sigForAll = TRUE,
  useFullNames = TRUE,
  ...
)

Arguments

x

output of enrichGO or gseGO

firstSigNodes

number of significant nodes (retangle nodes in the graph)

useInfo

additional info

sigForAll

if TRUE the score/p-value of all nodes in the DAG is shown, otherwise only score will be shown

useFullNames

logical

...

additional parameter of showSigOfNodes, please refer to topGO

Value

GO DAG graph
