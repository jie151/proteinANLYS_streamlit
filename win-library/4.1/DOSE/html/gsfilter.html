gsfilter

Description

filter enriched result by gene set size or gene count

Usage

gsfilter(x, by = "GSSize", min = NA, max = NA)

Arguments

x

instance of enrichResult or compareClusterResult

by

one of 'GSSize' or 'Count'

min

minimal size

max

maximal size

Value

update object
