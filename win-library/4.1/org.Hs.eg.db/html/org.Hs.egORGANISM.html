org.Hs.egORGANISM {org.Hs.eg.db}R Documentation

The Organism for org.Hs.eg

Description

org.Hs.egORGANISM is an R object that contains a single item: a character string that names the organism for which org.Hs.eg was built.

Details

Although the package name is suggestive of the organism for which it was built, org.Hs.egORGANISM provides a simple way to programmatically extract the organism name.
