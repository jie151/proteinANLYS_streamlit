Description

Modify Set Names

Usage

set_name(x, ...) <- value

Arguments

x

A combination matrix returned by make_comb_mat.

value

New set names.

...

Other arguments.

Examples

set.seed(123)
lt = list(a = sample(letters, 10),
          b = sample(letters, 15),
          c = sample(letters, 20))
m = make_comb_mat(lt)
set_name(m) = c("A", "B", "C")
