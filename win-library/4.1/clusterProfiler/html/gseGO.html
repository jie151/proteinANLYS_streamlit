gseGO {clusterProfiler}R Documentation

gseGO

Description

Gene Set Enrichment Analysis of Gene Ontology

Usage

gseGO(
  geneList,
  ont = "BP",
  OrgDb,
  keyType = "ENTREZID",
  exponent = 1,
  minGSSize = 10,
  maxGSSize = 500,
  eps = 1e-10,
  pvalueCutoff = 0.05,
  pAdjustMethod = "BH",
  verbose = TRUE,
  seed = FALSE,
  by = "fgsea",
  ...
)

Arguments

geneList

order ranked geneList

ont

one of "BP", "MF", and "CC" subontologies, or "ALL" for all three.

OrgDb

OrgDb

keyType

keytype of gene

exponent

weight of each step

minGSSize

minimal size of each geneSet for analyzing

maxGSSize

maximal size of genes annotated for testing

eps

This parameter sets the boundary for calculating the p value.

pvalueCutoff

pvalue Cutoff

pAdjustMethod

pvalue adjustment method

verbose

print message or not

seed

logical

by

one of 'fgsea' or 'DOSE'

...

other parameter

Value

gseaResult object

Author(s)

Yu Guangchuang
