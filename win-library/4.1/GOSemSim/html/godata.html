godata

Description

prepare GO DATA for measuring semantic similarity

Usage

godata(
  OrgDb = NULL,
  keytype = "ENTREZID",
  ont,
  computeIC = TRUE,
  processTCSS = FALSE,
  cutoff = NULL
)

Arguments

OrgDb

OrgDb object

keytype

keytype

ont

one of 'BP', 'MF', 'CC'

computeIC

logical, whether computer IC

processTCSS

logical, whether to process TCSS

cutoff

cutoff of TCSS

Value

GOSemSimDATA object
