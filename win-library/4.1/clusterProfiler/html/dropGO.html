dropGO

Description

drop GO term of specific level or specific terms (mostly too general).

Usage

dropGO(x, level = NULL, term = NULL)

Arguments

x

an instance of 'enrichResult' or 'compareClusterResult'

level

GO level

term

GO term

Value

modified version of x
