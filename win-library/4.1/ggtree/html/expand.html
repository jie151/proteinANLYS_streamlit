Description

expand collased clade

Usage

expand(tree_view = NULL, node)

Arguments

tree_view

tree view (i.e. the ggtree object). If tree_view is NULL, the last ggplot will be used.

node

internal node number to specify a clade. If NULL, using the whole tree

Value

tree view
