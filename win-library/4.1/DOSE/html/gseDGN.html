gseDGN {DOSE}R Documentation

DisGeNET Gene Set Enrichment Analysis

Description

perform gsea analysis

Usage

gseDGN(
  geneList,
  exponent = 1,
  minGSSize = 10,
  maxGSSize = 500,
  pvalueCutoff = 0.05,
  pAdjustMethod = "BH",
  verbose = TRUE,
  seed = FALSE,
  by = "fgsea",
  ...
)

Arguments

geneList

order ranked geneList

exponent

weight of each step

minGSSize

minimal size of each geneSet for analyzing

maxGSSize

maximal size of each geneSet for analyzing

pvalueCutoff

pvalue Cutoff

pAdjustMethod

p value adjustment method

verbose

print message or not

seed

logical

by

one of 'fgsea' or 'DOSE'

...

other parameter

Value

gseaResult object
